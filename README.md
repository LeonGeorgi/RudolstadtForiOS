The latest screenshots are generated by the manually triggered GitHub Actions workflow in `.github/workflows/ios-screenshot-tests.yml`. Successful runs publish the German iPhone gallery shown above to GitHub Pages and provide all German and English iPhone and iPad screenshots as downloadable workflow artifacts.

## Special Features
- Dark mode
- Search in lists
- Stage numbers
- Nearby stages
- In-app map
- Save events in-app
- Warning message for events that overlap with a saved event
- Rate artists
- Calculate the best possible schedule according to artist rating (shown as recommendations)
