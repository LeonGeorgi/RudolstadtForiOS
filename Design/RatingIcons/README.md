# Rating heart icon concepts

Three fixed rating glyphs intended to replace the current runtime composition of overlapping SF Symbols:

- `rating-heart-1.svg`
- `rating-heart-2.svg`
- `rating-heart-3.svg`

The icons share a 24 × 24 view box and use a single black fill, so they can later behave as monochrome template images. The two- and three-heart variants have deliberate transparent spacing between the hearts instead of a baked-in white outline. This keeps the separation effective on light, dark, tinted, and image backgrounds.

These files remain the review sources. Matching copies are currently installed in `Shared/Assets.xcassets` for test integration in the app.
